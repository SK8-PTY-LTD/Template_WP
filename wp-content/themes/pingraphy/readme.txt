== Description ==
Pingraphy is an inspired from Pinterest style gallery theme which is a simple way to showcase a beautiful photo collection and post formats. Pingraphy is responsive and includes an easy method for posting photos to a front page gallery.

* Responsive Layout
* Custom Colors
* Custom Header
* Post Formats
* The GPL v2.0 or later license. :) Use it to make something cool.

== Installation ==

1. Activate the theme
2. Go to the Theme Option page
3. Setup theme options

* Setup theme options
Appearance -> Theme Option

1. Upload logo
2. Upload Favicon
3. Select Header Skin (Pingraphy provides 9 flat color UIs for Header Skin)
4. Copyright Text
5. Enable/Disable theme info and WordPress powered by links
